
Тестовое задание (Книга контактов)
Создать SPA приложение - книга контактов, которое включает в себя список контактов и подробное описание каждого контакта.

Функционал

    Контакты должны быть загружены с использованием API https://randomuser.me только при первом запуске приложения. Далее приложение должно работать с localStorage.
    Отображение списка контактов.
    Подробное описание контакта.
    Редактирование контакта.
    Сортировка списка контактов по алфавиту.
    Группировка списка контактов по первой букве.
    Поиск контакта.

Доп. требования

    Приложение должно быть адаптивно сверстано.
    Приложение должно работать в Chrome, Firefox, IE последних версий.

Требования к технологиям
Использование angular (v1), приветствуется использование утилиты для сборки (webpack, grunt, gulp).

Оценивается качество кода и понятность интерфейса.
